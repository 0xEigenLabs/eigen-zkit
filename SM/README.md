# SM
State Machine zkVM prototype abstracted from [zkevm-proverjs](https://github.com/0xPolygonHermez/zkevm-proverjs/tree/main/pil).

# Example

```
npm run buildrom
npm run buildstoragerom
npm run genstarkstruct
node src/main.js -w circuits/
```
